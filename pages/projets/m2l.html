Maison Des Ligues (M2L)

Présentation & Contexte

La M2L est représentée par monsieur Sapin, son président actuel. Conscient de la visibilité qu'apportent les nouveaux systèmes d'information, il a souhaité bénéficier d'un site internet qui mettra en avant la M2L.

Monsieur Sapin cherche continuellement à améliorer et diversifier les services proposés par la maison des ligues afin d'accentuer sa dimension sociale. continuellement à améliorer et diversifier les services proposés par la maison des ligues afin d'accentuer sa dimension sociale. Il a donc souhaité bénéficier d'un outil de recherche de partenaires sportifs pour aider ses membres dans leur pratique quotidienne.


Solutions

Pour la création de ce projet, la solution était de développer leur site interne en PHP avec le Framework Cake PHP avec un serveur de Base de Données en PostgreSQL


Organisation

Pour l'organisation de ce projet en équipe, nous avons utilisé un tableau Trello de Atlassian pour se repérer dans les diverses tâches à réaliser et un Google Drive pour transmettre en équipe les fichiers.

Diagramme UML du Projet

Le diagramme compose les 7 pages demandées pour le Projet :

  • Catégorie
  • Championnat
  • Division
  • Type Championnat
  • Club
  • Equipe
  • Match



Un temps de réflexion a été nécéssaire tout au long du projet pour l'achever complétement. Notamment les relations entre entités Equipe & Match.
La cardinalité n'est pas comme les autres, mais d'autant plus logique : dans un seul match, il ne peut y avoir que 2 équipes au minimum mais aussi au maximum.




Installation de l'Environnement (PostgreSQL & Cake PHP)

Avant de démarrer quoi que ce soit, il faut installer l'environnement ainsi que le framework.

Utilisation de pgAdmin4 pour visualiser la base de données en mode graphique comme phpMyAdmin.
Admirer est aussi un outil de visualisation si c'est sur Wamp64.
Sinon, l'utilisation en ligne de commande pour visualiser avec Postgre est possible !

Installation du projet avec Composer (préalablement installé) pour Cake PHP



Création de la base de données

Une fois l'installation de toute l'environnement (Cake, Postgre), il faut pouvoir faire la base de données ainsi que la configuration nécéssaire pour connecter notre base de données dans CakePHP.

Détail du projet et rendu final

L'accueil principale du site avec un lien de redirection sur les 7 sections de la M2L.

La page division et catégorie.

Page d'édition d'une catégorie : on peut modifier son nom et le montant d'indémnité

Page Type Championnat

Page Championnat : la section regroupe les informations de catégorie, de division et de type de championnat

Lorsqu'un championnat est modifiable, les informations liés à ce championnat peuvent être modifiées.

Page Club

Page Equipe : le nom de l'équipe associé avec un club.

Page Match : cette page permet de gérer les matches entre deux équipes distinctes, la date et l'heure ainsi que le lieu.
