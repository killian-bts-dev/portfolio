Android - Quizz

Présentation & Contexte

Dans le but d'un projet d'android dans le cadre du BTS SIO, j'ai pu experimenter la création d'une application android avec le langage Kotlin. Le but était de créer une application quizz avec un schéma conceptuel, une base de données et une API (Application Programming Interface).


Diagramme UML du Projet

Le diagramme se compose de 7 classes du projet :

  • Question
  • Reponse
  • Theme
  • Questionnaire
  • Niveau
  • Vrai/Faux
  • QCM



Le projet est un système de Quiz qui se compose de questions avec 4 possibilités de réponses avec des niveaux et des thèmes différents.
Le questionnaire se compose de plusieurs questions qui peut être du type QCM ou un Vrai ou Faux, dans ce cas, 2 possibilités de réponses sont possibles.




Initialisation avec Android Studio

Avant de démarrer quoi que ce soit, il faut installer l'environnement ainsi que le framework.

Utilisation de pgAdmin4 pour visualiser la base de données en mode graphique comme phpMyAdmin.
Admirer est aussi un outil de visualisation si c'est sur Wamp64.
Sinon, l'utilisation en ligne de commande pour visualiser avec Postgre est possible !

Installation du projet avec Composer (préalablement installé) pour Cake PHP
